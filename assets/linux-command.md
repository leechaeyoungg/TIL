# linux command
>리눅스 명령어들을 정리해봅시다.

- pwd(print working directory)
    - 현재 작업중인 경로를 출력하는 명령어

- ls(list)
    - 현재 폴더에 있는 파일, 폴더(파란색)를 출력

- `-a`(optional) : 숨김처리된 파일, 폴더까지 출력해주는 명령어
- `..` 상위 폴더로 이동

- cd(change directory)
    - 폴더 이동시 사용

- mkdir(make directory)
    - 폴더 생성 명령어

- touch
    - 파일 생성 명령어

- rm(remove)
    - 파일 및 폴더 삭제
    - `-r`(optional) :폴더를 지우기 위해서 입력해야하는 옵션
